## Development notes

This project uses `ruff` for formatting and linting. For VS Code, just install
the "Ruff" extension and reload the editor.

### Frontend

```
cd frontend
nvm install 20
nvm use 20
npm install -g @vue/cli
npm install
npm run serve
```
